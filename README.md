# Ilustra

Proyecto de sitio web estático para presentar la historia, artistas y propuesta visual de Ilustra.

## Estructura del proyecto
- index.html: página principal y sección de bienvenida.
- artistas.html: página dedicada a los artistas.
- historia.html: página con la línea de tiempo histórica.
- css/index.css: estilos principales del sitio.
- js/history.js: lógica de la línea de tiempo.

## Convenciones de trabajo
- Se prioriza una estructura HTML clara y semántica.
- Los nombres de clase deben seguir la convención SUIT CSS.
- Todo cambio relevante debe documentarse en este archivo README.md.
- Todo cambio significativo debe registrarse en PLAN.md con fecha y resumen.

## Estado actual
- Se ha implementado la línea de tiempo histórica interactiva.
- Las tarjetas de artistas abren un lightbox con imagen y descripción al hacer clic o pulsar Enter.
- Se están aplicando convenciones de nombres consistentes en HTML y CSS.
